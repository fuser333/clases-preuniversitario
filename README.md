## 🚀 Día 9 · Streamlit Apps de Datos Interactivas

📂 `dia-09-streamlit/`

App web interactiva construida con Streamlit · convierte un análisis de datos en una página web que cualquiera puede usar desde el navegador.

**Para correr local:**
```bash
cd dia-09-streamlit
pip install -r requirements.txt
streamlit run app.py
```

**Para publicar gratis en internet:**
1. Forkea este repo
2. Ve a https://share.streamlit.io
3. Conecta este repo y selecciona `dia-09-streamlit/app.py` como main file
4. Listo · URL pública en 3 minutos
